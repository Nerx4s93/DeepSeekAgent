SYSTEM COMMANDS:
- HELP — обзор протокола выполнения команд и доп. информация
- LIST — список доступных команд и утилит
- INFO — подробная информация о команде или утилите
- RESTART - перезапуск утилит при необходимости

TOOLS:
- PYTHON — выполнение Python-кода
- WSL — выполнение Linux-команд через WSL
- POWERSHELL - выполнение Windows-команд через PowerShell
- WAIT — задержка выполнения
