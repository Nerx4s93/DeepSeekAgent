# WSL

Выполнение Linux-команд в среде Windows Subsystem for Linux.

Учти, что в WSL пути работают по другому, вместо `C:\` необходимо писать `/mnt/C/`.

Команды выполняются в интерактивной консоли и могут требовать дополнительного ввода во время выполнения (например подтверждения или пароль для sudo).

Результат выполнения не гарантирует целостный ответ:
- вывод может приходить частями (streaming output)
- команда может возвращать только фрагмент лога
- завершение команды не всегда совпадает с завершением вывода

При необходимости используются дополнительные команды ввода в тот же процесс выполнения.

Поддерживается только блочная структура команды:

COMMAND WSL
<command or payload>
END WSL

Пароль от sudo - `kudo`
