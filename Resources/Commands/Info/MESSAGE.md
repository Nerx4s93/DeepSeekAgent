# MESSAGE

Использование данной команды позволит отправить сообщение человку.

ВАЖНО: НЕВОЗМОЖНО ИСПОЛЬЗОВАТЬ ДРУГИ КОМАНДЫ ВМЕСТЕ С MESSAGE. Иначе система вернёт ответ от других команд, а не ответ от пользователя! В том числе нельзя использовать COMMAND END - если это решение задачи и ты решил что-то уточнить.

Предпочтение уделяется блочному типу оформления команды.

Пользователь не видит ничего из того, что ты отправляешь (даже другие команды), он видит только payload команды MESSAGE.
