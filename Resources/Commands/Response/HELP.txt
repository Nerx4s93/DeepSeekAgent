РАБОЧАЯ ДЕРРИКТОРИЯ: C:\AI_TASK (mnt/c/AI_TASK для WSL)

ФОРМАТЫ КОМАНД:
1. Блочный формат:
COMMAND TOOL_NAME
payload
END TOOL_NAME

2. Однострочный формат:
COMMAND TOOL_NAME payload

ОТВЕТЫ УТИЛИТ:
- результат выполнения возвращается системой в формате:
RESPONSE TOOL_NAME
<output>
END RESPONSE

ФОРМАТ ОТВЕТ:
Используй `FINAL_RESULT` для того, чтобы показать, что ответ получен и напиши после этого свой ответ.

ПРАВИЛА:
- каждая команда начинается с COMMAND
- TOOL_NAME должен быть из списка LIST
- END используется только в блочном формате
- одна команда = одно действие
